# PatternMatching

-На вход:
len  (длина массива char'ов)
m    (сам массив)
n    (число подстрок, которые нужно найти)
-далее n подстрок в формате: 
l - длина подстроки
str - сама подстрока

-На выходе:
n ответов в формате:
номер подстроки (в каком они были поданы на входе) - количество раз, которое она встретилась в строке
